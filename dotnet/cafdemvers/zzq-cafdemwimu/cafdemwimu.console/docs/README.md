# cafdemwimu.console — Documentation

This project is an **F# replication** of the C# project at
`dotnet/cafdemalihapa`. Classes are being ported folder-by-folder, preserving
the original namespace layout (with the root namespace changed from
`cafdemalihapa` to `cafdemwimu.console`).
